FAQ

Perguntas frequentes

Os dados são actualizados e precisos?
Os dados são recolhidos automaticamente de fontes públicas (OpenStreetMap, Google Maps, websites das empresas). São dados reais, mas podem ter imprecisões — um email pode estar desactualizado, um número pode ter mudado. Recomendamos sempre verificar antes de contactar.
Como é entregue o pack depois de pagar?
Após pagamento és redireccionado para a página de download. Os PDFs são gerados directamente no teu browser em segundos — sem esperar, sem email. Descarregas na hora.
Funciona fora dos Açores?
Neste momento cobrimos os Açores com mais de 1300 negócios mapeados. Expansão para o resto de Portugal está planeada. Para pedidos especiais, entra em contacto.
O que é o score de oportunidade?
Uma pontuação de 0 a 100. Quanto mais alto, mais gaps digitais a empresa tem e maior a probabilidade de precisar dos teus serviços. Acima de 70 é considerado alta oportunidade. Quanto mais próximo de 100, mais urgente a necessidade digital.
Posso seleccionar negócios de vários nichos?
Sim. No mapa podes seleccionar negócios de categorias diferentes na mesma compra — restaurantes, alojamento, salões, ginásios, o que precisares. A selecção é totalmente livre.
