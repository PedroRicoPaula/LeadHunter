2. Dados recolhidos

Recolhemos os seguintes dados pessoais:

  • ·Dados de pagamento: endereço de email, país de faturação e dados de pagamento processados pela Stripe. O Lead Hunter não armazena dados de cartão — são tratados exclusivamente pela Stripe.
  • ·Dados de encomenda: nicho e região solicitados, produto adquirido, data e valor da compra.
  • ·Comunicações: emails trocados para suporte ou entrega do produto.

3. Finalidade do tratamento

  • ·Processar e confirmar pagamentos
  • ·Entregar o produto (ficheiro CSV/PDF) ao email do cliente
  • ·Prestar suporte pós-venda
  • ·Cumprir obrigações legais e fiscais

Não utilizamos os dados para marketing sem consentimento explícito.

4. Base legal

O tratamento de dados é efectuado com base na execução do contrato (Art. 6.º, n.º 1, al. b) do RGPD) — necessário para entregar o produto adquirido — e no cumprimento de obrigações legais (Art. 6.º, n.º 1, al. c) do RGPD) para efeitos fiscais.

5. Partilha de dados com terceiros

Os dados são partilhados apenas com os seguintes subcontratantes, estritamente necessários para a prestação do serviço:

  • ·Stripe, Inc. — processamento de pagamentos. Política de privacidade: stripe.com/pt/privacy
  • ·Google (Gmail) — envio de email para entrega do produto.

Não vendemos, alugamos nem partilhamos dados pessoais com terceiros para fins comerciais.

6. Dados nos packs de leads

⚠️ Nota importante

Os ficheiros CSV/PDF entregues ao cliente contêm dados de empresas recolhidos de fontes públicas (OpenStreetMap, TripAdvisor, websites públicos das empresas). Estes dados referem-se a pessoas colectivas (empresas) e não a pessoas singulares, pelo que na sua maioria não constituem dados pessoais nos termos do RGPD.

Na eventualidade de algum dado corresponder a uma pessoa singular identificável (ex: nome de um empresário em nome individual), o cliente é responsável pelo tratamento desses dados de acordo com a legislação aplicável.

7. Conservação de dados

Os dados de encomenda e faturação são conservados pelo período mínimo exigido por lei para obrigações fiscais (10 anos em Portugal). Os emails de suporte são eliminados após resolução da questão ou no prazo máximo de 2 anos.

8. Os seus direitos (RGPD)

Enquanto titular de dados, tem os seguintes direitos:

  • ·Acesso — saber que dados temos sobre si
  • ·Rectificação — corrigir dados inexactos
  • ·Apagamento — solicitar a eliminação dos seus dados (salvo obrigações legais)
  • ·Portabilidade — receber os seus dados em formato estruturado
  • ·Oposição — opor-se ao tratamento em determinadas circunstâncias

Para exercer qualquer destes direitos, contacte: pedroricopaula@gmail.com. Respondemos no prazo de 30 dias.
